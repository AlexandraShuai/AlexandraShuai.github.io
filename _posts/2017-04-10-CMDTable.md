## 常用命令速查
## shell
```ruby
//切换shell
chsh -s /bin/zsh
```
## rvm
```ruby
//获取最新rvm版本
rvm get head

//获取稳定rvm版本
rvm get stable
```

## gitlab-ci
```ruby
//重启
gitlab-ci-mutil-runner restart

//开始让runner在系统权限模式下跑起来，--debug可打印日志
sudo gitlab-ci-multi-runner --debug run

```

## Xcode
```ruby
//查看Xcode内已安装的所有模拟器列表
xcrun simctl list devices
```
